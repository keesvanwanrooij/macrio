# Contributing to Macrio

Thanks for helping build a community-owned food & fitness tracker!

## Ways to contribute

- **Code:** pick an issue, branch (`feat/<name>`), open a PR. See `docs/process/BRANCHING.md`.
- **Translations:** edit `app/src/locales/*.json`, follow `docs/i18n/GLOSSARY.md`.
- **Food data:** use the app — creating and improving products *is* contributing.
- **Feedback:** in-app feedback form or GitHub issues (templates provided).

## Ground rules

- Follow the `CODE_OF_CONDUCT.md`.
- Every PR: TypeScript clean, both languages covered, RLS respected — see `docs/process/DEFINITION_OF_DONE.md`.
- Conventional Commits (`feat:`, `fix:`, `docs:`...).
- By contributing product data you agree it is released under ODbL-compatible terms (keeps the database open).
