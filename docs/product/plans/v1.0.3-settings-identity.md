## Problem

Username typos are stuck; DOB is awkward to type; no app version for support; date format should be a preference.

## Goal

Users can fix username, pick DOB from a calendar, choose date format, and see the app version in Settings.

## In scope

- Edit username (unique + format rules)
- App version in Settings
- DOB calendar picker (onboarding + profile)
- Date format setting applied to displayed/entered dates

## Out of scope

- Full name changes (already optional elsewhere)
- Changing email (unless already trivial via Supabase)

## Acceptance criteria

- [ ] Username update fails clearly on collision / invalid format
- [ ] DOB picker works on Android (Expo Go)
- [ ] Date format setting persists on profile
- [ ] Version string matches `app.json` / Expo config
- [ ] nl/en via i18n

## Open questions
