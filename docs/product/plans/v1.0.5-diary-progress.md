## Problem

Only day-level totals; hard to see progress toward goals while logging.

## Goal

See macros **per meal** and **progress toward daily goals** on the diary.

## In scope

- Per-meal macro totals
- Daily progress UI for kcal + macros (respect count direction + focus/overview)

## Out of scope

- Custom report periods (v1.3)

## Acceptance criteria

- [ ] Each meal block shows its summed kcal/macros
- [ ] Progress visible for set goals; sensible empty state if no goals
- [ ] Works with count up and count down
- [ ] nl/en via i18n
- [x] Week reports: macro chart + ghost empty days + historical goal line/ticks
- [x] Day reports: swipe totals/diary → macros; swipe elsewhere → day; swipe Day/Week switcher → toggle mode
- [x] Week: swipe chart/average → macros; swipe elsewhere → week; tap bar → diary
- [x] Goal saves write `goal_revisions` for today (migration `014`)

## Open questions

- Bars vs compact numeric remaining/eaten? (header bars shipped; meal-level still open)
