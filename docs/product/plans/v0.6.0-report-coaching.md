## Problem

Reports show numbers but little plain-language feedback. Short coaching lines improve clarity and retention without LLM cost/risk.

## Goal

Under day (and week where natural) reports, show a small set of **rule-based** lines in nl/en.

## Example (NL)

```
Vandaag:
✅ 182g eiwit (102%)
⚠️ 340 kcal onder doel
✅ Mooie macroverdeling
💡 Nog 18g vet beschikbaar
```

## In scope

- Rules from totals vs goals (hit / under / over), simple macro balance heuristics
- i18n strings; icons/emoji consistent with existing UI tone
- Works when some goals are unset (skip those lines)

## Out of scope

- Generative AI / personalized coach (v2.0)
- Allergen-topic coaching (later with allergen reports)

## Acceptance criteria

- [ ] Day report shows coaching block when enough data exists
- [ ] nl + en; no network call for the text
