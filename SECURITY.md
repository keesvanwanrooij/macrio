# Security Policy

## Reporting a vulnerability

Please report vulnerabilities privately via [GitHub Security Advisories](https://github.com/keesvanwanrooij/macrio/security/advisories/new) — do not open public issues for security problems.

You can expect an initial response within 7 days. Please include reproduction steps and affected versions.

## Scope

- The Macrio mobile app and Supabase backend configuration in this repo.
- Especially interesting: RLS bypasses, auth flaws, access to other users' diary/health data.

## Supported versions

Only the latest released minor version receives security fixes.
